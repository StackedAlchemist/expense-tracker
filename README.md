# 💰 Arcane Ledger — Expense Tracker

> *"Track your mana flow. Know where every coin is spent."*

A clean, themeed personal expense tracker built with vanilla HTML, CSS, and JavaScript. Log income, expenses, and borrowed funds — then filter, sort, and visualize your spending at a glance.

**[🔮 View Live Demo](https://stackedalchemist.github.io/stacked-alchemist/expense-tracker/)**

---

## ✦ Features

- **Three Transaction Types** — Income, Expense, and Borrowed (tracks money owed separately)
- **Category System** — Food & Drink, Housing, Transport, Entertainment, Health, Shopping, Work, Savings, and more
- **Spending Progress Bar** — Visual indicator of what percentage of income has been spent
- **Filter & Sort** — Filter by type or category, sort by newest, oldest, highest, or lowest amount
- **Edit & Delete** — Full CRUD on every transaction via modal editor
- **Persistent Storage** — All data saved to localStorage, survives page refreshes
- **Summary Cards** — At-a-glance totals for income, expenses, and current balance
- **Custom Cursor** — Arcane-themed cursor with hover effects

## 🚀 Running Locally

No dependencies or build tools required.

```bash
git clone https://github.com/StackedAlchemist/stacked-alchemist.git
cd Portfolio/expense-tracker
# Open index.html in your browser
```

---

## ⚠ Data Storage Notice

> The archives are bound to this vessel. Data persists in local memory only — switching devices or clearing your browser cache will release the binding.

All transaction data is saved to **localStorage** in the user's browser. No account or backend required. Clearing browser data will erase saved transactions.
